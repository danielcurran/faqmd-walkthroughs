The largest and most commonly used menu is the Party Menu. This is accessed by pressing 'A' in the field, and contains seven main options: ITEM - Opens the item list. From here you can select items to be used, examined or discarded. TECH - Opens another small menu of the characters. Select one and a list of that character's usable Techniques will appear. From here you can use curative Techniques and travel Techinques. SKILL - Opens another small menu of the characters. Select one and a list of that character's usable Skills will appear. Opens another small menu of the characters. Select one to view all information about the character. ORDER - Opens another small menu of the characters. From here you can set their marching order by selecting the characters in your preferred sequence. It is worth noting that those characters at the top of the list are more likely to be attacked in battle than those at the bottom. TALK - This starts a small dialogue between the members of your party, allowing you to recap on the current situation and to give you clues as to where to go next.

Opens a list consisting of the letters A to H. These represent the eight Macros that may be set. Macros are sets of actions that can be pre-programmed for use in battle. By defaul, the Macro A is in white, and this means that it has been assigned and that it is usable. Initially, Macro A is set to Attack for all characters, and it is probably best to leave it that way. You can set a Macro by selecting one marked in red (unset). A list of characters appears. Select the first character you want to perform an action, and then a menu, similar to the battle command menu, appears. Choose an action as you would in battle, and then repeat the process for each character to create the Macro. This Macro will then be marked in white. Selecting a Macro that is already assigned will allow you to delete a Macro. Use this if the Macro involves characters not in your party or if you need more space. If a Macro is marked in yellow it means that the Macro will not be able to be carried out as specified. This occurs when a character is set to perform an action that they are unable to perform. In this case a Macro will still be run as instructed, but characters marked in red on the battle Macro screen will do nothing. 4. Terminology CTMLG
